概要

Wreckは、Discord上で簡単にメッセージを大量送信できるRaid Botです。 アプリ連携のみで使用可能で、サーバー導入が不要なため、手軽に使うことができます。 @everyone、@hereなどのメンションも使用可能で、荒らし行為を容易に行うことができます。

注意

このボットの使用に伴う責任は全てユーザーにあります。不適切な使用によるアカウント停止などの責任は負いかねます。

使い方

1

アプリの追加

以下のリンクを開き、「アプリを追加」を押し、アプリを追加してください。

https://discord.com/oauth2/authorize?client_id=1348798218565189642
2

コマンドの実行

荒らしたいサーバーに行きコマンドを打つことで、実行できます。

3

実行上の注意点

  • サーバーに入ってから最低でも1日以上経ってから荒らすことを推奨
  • ステータスをオフラインにして使用する
  • 自己責任で使用すること

コマンド一覧

/spam

メッセージを連続して送信することができます。このコマンドを打つとフォームが開き、メッセージ内容やメンション設定ができます。

/register slot:[1-5]

カスタムメッセージを登録します。スロット1～5に保存できます。

/custom-message[1-5]

登録したカスタムメッセージを送信します。

/help

コマンドの使い方などを表示します。

メリット・デメリット

メリット

  • @everyone, @hereなどのメンションが使える (大半のサーバーは無理)
  • アプリ連携のみで使え、サーバー導入が不必要
  • ボットがメッセージを送るので、匿名化
  • 外部アプリの使用はデフォルトで許可されているため、大半のサーバーで荒らしは可能

デメリット

  • APIの性質上、一回のリクエストにつき6件までしかメッセージを送れない
  • アカウントをバンされるとメッセージが消える

セットアップ

必要条件

  • Node.js 16.9.0以上
  • Discord Bot Token (Discord Developer Portalから取得)

インストール手順

このリポジトリをクローンし、wreckへディレクトリに移動

git clone https://github.com/n4n45h1/wreck.git
cd wreck

npmを使用する場合

npm install
npm start

yarnを使用する場合

yarn install
yarn start
